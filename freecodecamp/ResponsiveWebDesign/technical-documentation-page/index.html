Basic Javascript

Javascript is the langugage of the web. When you want to make your website interactive,Javascript is the way to do it. With JavaScript you can change the webpages by getting an element, like an button, and say onclick, add 10 points to intellect to your roleplaying character. With additional function to get bonus stats in charisma for every 20th point.

Here is an example of a simple function where intellect is set to 10. And at level up an additional 10 is placed in intellect, resulting in 20 int and +10 bonus to charisma. 

function levelUp(intExperience) {
                            let intellect = 10;
                            console.log("intellect: " + intellect)
                            intExperience = intellect += 10;
                            let charisma = 20;
                            if (intellect === 20) {
                                charisma += 10;
                                console.log("+10 intellect! Total, " + intellect);
                                console.log("+10 bonus charisma! Total, " + charisma);
                                return;
                            };
                            intellect += 10;
                            console.log("+10 intellect! Total, " + intellect);
                        };

When I run levelUp(10) the output in the console.log message will be, 

// result in console.log:
                        intellect: 10
                        +10 intellect! Total, 20
                        +10 bonus charisma! Total, 30

ECMAScript 6

With es6 came a lot of new features...

Here are just a few good to know features explained:

  • Template literals
  • Arrow function expressions
  • let and const
  • Spread and Rest syntaxes
  • Classes

Template literals:

The ability to write javascript with HTML code. For example you can write, `Charisma level is: ${charisma}` to get the charisma value within the paragraph text.

Arrow function expressions:

Here is the difference between a simple addition function. 

function add(num1, num2) {
                        return num1 + num2;
                    }

With arrow functions it looks like,

const add = (num1, num2) => a + b;

With the shorter syntax it helps clean up the code and run immedieatly without return key word, it has a lot of benefits, but also some drawbacks such as changing how .this keyword behaves and can't be a constructor.

let and const!

With the addition of the ability to declare variables with let and const, it changed from using var avocado = 3 to let avocado = 3. Which might not seem like a big change. But the var keyword is a global variable, which is accesable for the entire page, also called "global scope". While let and const is in block scope. Which simply is only avalible inside their function code. Example: 

function add(num1, num2) {
                        var num1 = 10;
                        var num2 = 20;
                        var result = num1 + num2;
                        return result;
                    }

num1 and num2 are both avalible outside of block scope. If I were to add it outside, it would update inside the function. Which isn't so bad if you are aware of it. But once you have multiple functions, you might end up overwritting one of them resulting in wrong calculations.

If instead you would use let (const is used with constant variables), it would only be avalible in the function scope, which means you could use it again for a similair function in another part of your code without overwrtiing your earlier function.

Spread and Rest syntaxes (…)

Helps you copy an entire array to another array or make it easier to sum numbers together. Here's an example, 

                    function sum(num1, num2, num3) {
                        return num1 + num2 + num3;
                    }

                    const numbers = [10, 20, 30];

                    console.log(sum(...numbers));
                    // expected result: 60

Spread also makes it easier to copy an array of items to a new array like this,: 

                    const fruitBasket = ["Banana", "Apple", "Orange"];
                    const shoppingCart = ["Pizza", "Soda", ...fruitBasket];
                    // Output: (5) ['Pizza', 'Soda', 'Banana', 'Apple', 'Orange']
